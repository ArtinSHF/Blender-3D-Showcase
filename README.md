# Blender 3D Showcase 🎬

This repository is a showcase of some of my Blender projects, including 3D models, animations, and rendered scenes.

## 🧠 About

I use Blender to create different types of 3D projects, usually based on ideas I get and want to turn into a visual scene or animation.

Some examples of the types of projects I make include:

- Cars racing
- Fighter jets flying over a city
- Cinematic scenes
- 3D models
- Short animations

I sometimes edit my Blender animations in CapCut and upload them to YouTube.

## 🛠 Tools Used

- Blender
- CapCut
- Photoshop / After Effects
- AI assistance
